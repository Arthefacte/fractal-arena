# Fractal Arena

**3v3 idle auto-combat game on Fractal Bitcoin**

Deploy your creatures, set your bet, and let them fight autonomously. Earn $FRACTALARENA tokens. Built by a solo dev on Fractal Bitcoin.

## Play

- **Browser** → [fractalarena.itch.io/fractalarena](https://fractalarena.itch.io/fractalarena)
- **Windows** → Download the latest `.exe` in [Releases](https://github.com/Arthefacte/fractal-arena/releases)
